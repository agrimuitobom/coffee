# 日本刀風Webサイトの設定方法

このリポジトリには、日本の包丁ブランドをイメージした美しいWebサイトのテンプレートが含まれています。以下の手順に従って、サイトを設定して開始してください。

## ファイル構造

```
knife-website/
│
├── index.html        # HTMLの構造を含むメインファイル
├── styles.css        # すべてのスタイリングを含むCSSファイル
└── script.js         # インタラクティブな機能を含むJavaScriptファイル
```

## 機能

このテンプレートには以下の機能が含まれています：

- 美しいミニマルデザイン
- モバイルレスポンシブレイアウト
- フルスクリーンナビゲーションメニュー
- スムーズスクロール
- スクロール時のヘッダー変換効果
- 縦書きテキストのサポート（和風デザインに最適）
- フェードインアニメーション
- 遅延読み込み（Lazy Loading）のサポート
- 波紋パターン（包丁のダマスカス模様風）のスタイル
- 商品ギャラリー機能

## HTMLの構造

HTMLファイルは次のセクションに分かれています：

1. **ヘッダー** - ロゴとナビゲーションを含む
2. **フルスクリーンメニュー** - モバイルでの表示用
3. **ヒーローセクション** - メイン画像と見出し
4. **季節のコンテンツセクション** - 日本の四季をテーマにしたコンテンツ
5. **商品紹介セクション** - 商品の画像と説明
6. **フッター** - ナビゲーションリンクとソーシャルアイコン

## CSSのカスタマイズ

`styles.css`ファイルでは、以下のカスタマイズが可能です：

### カラースキーム

```css
:root {
    --primary-font: 'Noto Serif JP', serif;
    --secondary-font: 'Noto Sans JP', sans-serif;
    --text-color: #333;
    --light-bg: #f9f9f9;
    --dark-bg: #222;
    --accent-color: #a67c52;
}
```

これらの変数を変更することで、サイト全体のカラースキームを簡単に変更できます。

### アニメーションとエフェクト

フェードインアニメーションを使用するには、要素に`.fade-in`クラスを追加します：

```html
<div class="fade-in">
    アニメーションさせたいコンテンツ
</div>
```

縦書きテキストを使用するには、要素に`.vertical-text`クラスを追加します：

```html
<p class="vertical-text">
    縦書きで表示するテキスト
</p>
```

波紋パターン（ダマスカス模様風）を適用するには、要素に`.damascus-pattern`クラスを追加します：

```html
<div class="damascus-pattern">
    波紋パターンを適用するコンテンツ
</div>
```

## JavaScriptの機能

`script.js`ファイルには以下の機能が実装されています：

1. **メニュー開閉の制御** - ハンバーガーメニューのクリックでフルスクリーンメニューを表示
2. **ヘッダースクロールエフェクト** - スクロール時にヘッダーの背景とスタイルを変更
3. **スムーズスクロール** - ページ内リンクをクリックした際に滑らかにスクロール
4. **画像の遅延読み込み** - パフォーマンス向上のため、必要なときだけ画像を読み込む
5. **フェードインアニメーション** - 要素がビューポートに入ったときにフェードイン表示

## 使用方法

1. 3つのファイル（`index.html`、`styles.css`、`script.js`）をダウンロードする
2. HTMLファイルを編集して、自分のブランドやコンテンツに合わせる
3. 必要に応じてCSSをカスタマイズする
4. Webサーバーにファイルをアップロードするか、ローカルで開く

## 画像の追加

サイトに画像を追加するには、プロジェクトに`images`フォルダを作成し、そこに画像を配置します。その後、HTMLファイル内の画像パスを更新します：

```html
<img src="images/your-image.jpg" alt="画像の説明">
```

遅延読み込みを使用するには：

```html
<img data-src="images/your-image.jpg" alt="画像の説明" class="lazy-load">
```

## モバイル対応

このテンプレートはモバイルファーストでデザインされており、すべての画面サイズで適切に表示されます。レスポンシブデザインは`styles.css`内のメディアクエリで制御されています。

## カスタマイズのヒント

- フォントをカスタマイズするには、Google Fontsから別の日本語フォントを選び、HTMLのhead部分で読み込みを変更し、CSSの変数を更新します。
- 色のカスタマイズは、CSSのroot変数を変更するだけで簡単に行えます。
- 画像は高品質なものを使用し、必要に応じて最適化してパフォーマンスを向上させてください。
- 商品写真は白または薄いグレーの背景で撮影されたものが日本の伝統的な美学と調和します。
